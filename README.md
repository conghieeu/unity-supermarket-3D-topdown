# Unity Supermarket 3D Top-down

Dự án mô phỏng quản lý siêu thị góc nhìn từ trên xuống (Top-down) được phát triển bằng công cụ Unity. Người chơi sẽ điều khiển nhân vật thực hiện các hoạt động vận hành siêu thị như sắp xếp hàng hóa, phục vụ khách hàng và mở rộng kinh doanh.

## 🚀 Tính năng chính

- **Góc nhìn Top-down 3D:** Trải nghiệm quan sát toàn cảnh siêu thị trực quan.
- **Hệ thống nhân vật:** Điều khiển nhân vật di chuyển và tương tác với các vật phẩm trong môi trường.
- **Quản lý hàng hóa:** Nhập kho, trưng bày sản phẩm lên kệ và kiểm soát số lượng.
- **Cơ chế kinh doanh:** Hệ thống thanh toán và phục vụ khách hàng (đang phát triển).

## 🛠 Công nghệ sử dụng

- **Engine:** Unity (phiên bản đề xuất: 2022.x hoặc mới hơn)
- **Ngôn ngữ:** C#
- **Render Pipeline:** Built-in hoặc URP (tùy thuộc vào cấu hình Assets)
- **Editor:** Visual Studio / VS Code

## 📁 Cấu trúc dự án

- `/Assets`: Chứa toàn bộ mã nguồn (Scripts), mô hình 3D (Models), Prefabs và Scenes.
- `/Builds`: Chứa các bản build thực thi của trò chơi.
- `/ProjectSettings`: Cấu hình hệ thống của Unity.
